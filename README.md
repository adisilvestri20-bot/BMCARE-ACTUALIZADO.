## Cómo publicar en GitHub Pages (paso a paso)

1. Crea una cuenta en https://github.com y un **repositorio nuevo** (marca **Public**).
2. **Add file → Upload files** y arrastra **`index.html`** al repositorio.
3. **Commit changes**.
4. Ve a **Settings → Pages**. En *Branch* elige **main** y carpeta **/(root)**. **Save**.
5. Espera 1–2 minutos y recarga. Tu sitio quedará en:
   `https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`

### Si te sigue saliendo error 404
- Asegúrate de que el archivo se llame exactamente **`index.html`** (en minúsculas) y esté en la **raíz** del repositorio, no dentro de una subcarpeta.
- En **Settings → Pages** confirma que diga *"Your site is live at…"*. Si recién lo activaste, espera unos minutos.
- Recarga con Ctrl+F5.
